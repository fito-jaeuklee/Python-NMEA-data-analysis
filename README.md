# Python-NMEA-data-analysis

### 1) 개발환경 설정

micropyGPS ( https://github.com/inmcm/micropyGPS ) 라는 사이트의 파싱 API를 사용하였기 때문에
관련 설치를 진행해야한다.

```
python setup.py install
```

### 2) 사용 방법

사용자 기준에서 사용해야할 파일은 NMEA_value_save_main 이다.
